# 📱 Как установить приложение на Android телефон

## Способ 1: Через Android Studio (рекомендуется для разработки)

### Шаг 1: Включите режим разработчика на телефоне
1. Откройте **Настройки** → **О телефоне**
2. Найдите **Номер сборки** (может быть в разделе "О телефоне" или "Сведения о телефоне")
3. Нажмите на **Номер сборки** 7 раз подряд
4. Появится сообщение "Вы стали разработчиком!"

### Шаг 2: Включите отладку по USB
1. Откройте **Настройки** → **Для разработчиков** (или **Параметры разработчика**)
2. Включите **Отладка по USB**
3. (Опционально) Включите **Оставаться активным при подключении**

### Шаг 3: Подключите телефон к компьютеру
1. Подключите телефон к компьютеру через USB кабель
2. На телефоне появится запрос "Разрешить отладку по USB?" → нажмите **Разрешить**
3. Отметьте галочку **Всегда разрешать с этого компьютера** (если хотите)

### Шаг 4: Запустите приложение из Android Studio
1. Откройте проект в Android Studio
2. В верхней панели выберите ваше устройство из списка устройств
3. Нажмите кнопку **Run** (▶️) или `Shift + F10`
4. Приложение установится и запустится на телефоне

## Способ 2: Установка APK файла вручную

### Шаг 1: Соберите APK файл

#### Вариант A: Через Android Studio
1. Откройте **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. Дождитесь завершения сборки
3. Нажмите **locate** в уведомлении или найдите файл в:
   ```
   app/build/outputs/apk/debug/app-debug.apk
   ```

#### Вариант B: Через командную строку
```bash
# Перейдите в корневую папку проекта
cd C:\Users\1\AndroidStudioProjects\Kleos

# Соберите Debug APK
.\gradlew assembleDebug

# APK будет в: app\build\outputs\apk\debug\app-debug.apk
```

### Шаг 2: Передайте APK на телефон
- **Через USB:** Скопируйте файл `app-debug.apk` на телефон
- **Через облако:** Загрузите в Google Drive/Dropbox и скачайте на телефон
- **Через email:** Отправьте себе на email и откройте на телефоне

### Шаг 3: Установите APK на телефоне
1. Откройте **Файлы** (или **Проводник**) на телефоне
2. Найдите файл `app-debug.apk`
3. Нажмите на него
4. Если появится предупреждение "Установка заблокирована", нажмите **Настройки** и включите **Разрешить установку из этого источника**
5. Нажмите **Установить**
6. После установки нажмите **Открыть**

## Способ 3: Установка через ADB (для продвинутых)

### Шаг 1: Установите ADB (если еще не установлен)
ADB обычно идет вместе с Android Studio. Путь:
```
C:\Users\1\AppData\Local\Android\Sdk\platform-tools\adb.exe
```

### Шаг 2: Включите отладку по USB на телефоне
(См. Способ 1, Шаги 1-2)

### Шаг 3: Подключите телефон и установите APK
```bash
# Перейдите в папку с ADB
cd C:\Users\1\AppData\Local\Android\Sdk\platform-tools

# Проверьте подключение
adb devices

# Если устройство видно, установите APK
adb install "C:\Users\1\AndroidStudioProjects\Kleos\app\build\outputs\apk\debug\app-debug.apk"
```

## ⚠️ Важные замечания

### Безопасность
- **Debug APK** не подписан и предназначен только для тестирования
- Для публикации в Google Play нужен **Release AAB** (Android App Bundle)

### Разрешения
- При первой установке APK Android может запросить разрешение на установку из неизвестных источников
- Это нормально для разработки и тестирования

### Проблемы и решения

**Проблема:** "Установка заблокирована"
- **Решение:** Включите "Установка из неизвестных источников" в настройках

**Проблема:** "Приложение не устанавливается"
- **Решение:** Удалите старую версию приложения (если была установлена)

**Проблема:** "ADB не видит устройство"
- **Решение:** 
  - Установите драйверы USB для вашего телефона
  - Попробуйте другой USB кабель
  - Включите "Передача файлов" вместо "Только зарядка"

**Проблема:** "Приложение крашится при запуске"
- **Решение:** 
  - Проверьте логи: `adb logcat`
  - Убедитесь, что телефон поддерживает минимальную версию Android (API 24 = Android 7.0)

## 📋 Быстрая инструкция (краткая версия)

1. **Включите режим разработчика** (Настройки → О телефоне → Номер сборки × 7)
2. **Включите отладку по USB** (Настройки → Для разработчиков)
3. **Подключите телефон к компьютеру** через USB
4. **В Android Studio:** Выберите устройство → Нажмите Run (▶️)

Или:

1. **Соберите APK:** Build → Build APK(s)
2. **Скопируйте APK на телефон**
3. **Откройте APK на телефоне** → Установить

---

## 🎯 Рекомендации

- **Для разработки:** Используйте Способ 1 (через Android Studio) - самый удобный
- **Для тестирования:** Используйте Способ 2 (APK файл) - можно установить на несколько устройств
- **Для автоматизации:** Используйте Способ 3 (ADB) - для CI/CD

---

**Готово!** Приложение должно установиться и запуститься на вашем телефоне. 🎉
