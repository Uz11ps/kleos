# Обновление цветовой схемы iOS приложения

## Выполненные изменения

Все синие цвета заменены на фиолетовые/пурпурные/розовые оттенки согласно Android версии.

### Основные цвета (добавлены в UIUtils.swift):

1. **kleosPurple** - `#8B5CF6` - Пурпурный (заменяет синий)
2. **kleosCyan** - `#06B6D4` - Циан
3. **kleosYellow** - `#FFD700` - Желтый/Золотой (для категории "Interesting")
4. **kleosPinkAccent** - `#FF6B9D` - Розовый акцент (для FAQ)
5. **kleosLavender** - `#E8D5FF` - Лавандовый (для категории "News")
6. **kleosLightPurple** - `#D4A5FF` - Светло-фиолетовый
7. **kleosLightPink** - `#FFB6C1` - Розовый (Light Pink)

### Цвета с прозрачностью:

1. **kleosDialogBg** - `#240A24` с opacity 0.64 - Фон диалогов
2. **kleosDialogTextBg** - `#EDC9EA` с opacity 0.25 - Фон текстовых блоков
3. **kleosDialogBorder** - `#886184` - Граница диалогов

### Обновленные файлы:

1. **UIUtils.swift**:
   - Добавлены все новые цвета
   - Обновлен CategoryBadge: News - лавандовый, Interesting - желтый
   - Обновлены размытые круги на фоне (теперь пурпурные)

2. **Все View файлы**:
   - Заменены все использования `Color.kleosBlue` на `Color.kleosPurple`
   - Заменены все использования `.blue` на соответствующие цвета из палитры
   - Обновлены кнопки отправки сообщений (пурпурные)
   - Обновлены ссылки и акценты (розовый акцент для FAQ)

3. **MainTabView.swift**:
   - Акцентный цвет TabBar изменен на пурпурный

### Цветовая схема по экранам:

- **Фон всех экранов**: `#0E080F` (темно-фиолетовый)
- **Размытые круги**: Пурпурный (`#8B5CF6`) с прозрачностью
- **Категория "News"**: Лавандовый (`#E8D5FF`)
- **Категория "Interesting"**: Желтый (`#FFD700`)
- **Кнопки и акценты**: Пурпурный (`#8B5CF6`)
- **FAQ акценты**: Розовый (`#FF6B9D`)

### Следующие шаги:

1. Пересоберите проект в Xcode (Product → Clean Build Folder, затем Build)
2. Запустите приложение и проверьте все экраны
3. Убедитесь, что цвета соответствуют Android версии
