## Новые файлы

1. **GradientShape.swift** - компонент градиентной формы для онбординга и Welcome
2. **FilterFields.swift** - компоненты для полей фильтров (FilterInputField, FilterPickerField)

## Что нужно сделать

1. **Пересоберите проект в Xcode:**
   - Product → Clean Build Folder (Cmd + Shift + K)
   - Product → Build (Cmd + B)
   - Product → Run (Cmd + R)

2. **Убедитесь, что все файлы добавлены в проект:**
   - `ios_app/Kleos/Utils/GradientShape.swift`
   - `ios_app/Kleos/Utils/FilterFields.swift`

3. **Проверьте все экраны:**
   - Онбординг должен показывать градиентные формы и текст
   - Welcome экран должен иметь градиентные формы
   - Главная страница должна иметь табы на русском и правильные карточки
   - Фильтры программ должны иметь темные поля с иконками

## Важные заметки

- Все экраны теперь используют единый NavigationView из MainTabView
- Цвета соответствуют Android версии
- Карточки новостей имеют правильные размеры и цвета
- Фильтры программ имеют правильный дизайн с иконками
