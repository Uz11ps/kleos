# Инструкция по пересборке проекта после изменений

## Важно: После изменений нужно пересобрать проект!

### Шаг 1: Откройте проект в Xcode
1. Откройте Xcode
2. Откройте файл проекта: `kleos.xcodeproj` (или создайте новый проект если его нет)

### Шаг 2: Убедитесь, что файл InputMasks.swift добавлен в проект
1. В левой панели (Project Navigator) найдите папку `Utils`
2. Проверьте, что там есть файл `InputMasks.swift`
3. Если файла нет:
   - Правой кнопкой на папку `Utils` → "Add Files to Kleos..."
   - Выберите файл `ios_app/Kleos/Utils/InputMasks.swift`
   - Убедитесь, что галочка "Add to targets: Kleos" стоит

### Шаг 3: Очистите кеш и пересоберите
1. В Xcode: **Product → Clean Build Folder** (или `Cmd + Shift + K`)
2. Подождите завершения очистки
3. **Product → Build** (или `Cmd + B`) - проверьте, что нет ошибок компиляции
4. Если есть ошибки - исправьте их

### Шаг 4: Запустите приложение заново
1. Остановите текущий запуск (если запущено)
2. **Product → Run** (или `Cmd + R`)
3. Дождитесь полной пересборки и запуска

### Шаг 5: Проверьте изменения
1. Откройте экран **Admission** - должны быть маски для телефона и дат
2. Откройте экран **Support** - должны быть все 30 FAQ вопросов, кнопка "Submit request" вверху
3. Откройте любую программу - должен быть новый дизайн с карточкой
4. Проверьте фон приложения - должен быть темнее (#0E080F)

## Если изменения не применились:

1. **Удалите приложение с устройства/симулятора** полностью
2. **Очистите Derived Data:**
   - Xcode → Preferences → Locations
   - Нажмите стрелку рядом с путем к Derived Data
   - Удалите папку для вашего проекта
3. **Пересоберите проект** заново (шаги 3-4)
