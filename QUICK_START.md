# 🚀 Быстрый запуск приложения на телефоне

## Шаг 1: Настройте телефон (один раз)

1. **Откройте Настройки** → **О телефоне**
2. Найдите **"Номер сборки"** (может быть в разделе "Сведения о телефоне" или "О телефоне")
3. **Нажмите на "Номер сборки" 7 раз подряд**
4. Появится сообщение **"Вы стали разработчиком!"**

5. Вернитесь в **Настройки** → теперь появится раздел **"Для разработчиков"** (или **"Параметры разработчика"**)
6. Откройте его и включите:
   - ✅ **"Отладка по USB"**
   - (Опционально) **"Оставаться активным при подключении"**

## Шаг 2: Подключите телефон к компьютеру

1. Подключите телефон к компьютеру через **USB кабель**
2. На телефоне появится запрос **"Разрешить отладку по USB?"**
3. Нажмите **"Разрешить"**
4. Отметьте галочку **"Всегда разрешать с этого компьютера"** (если хотите)
5. Выберите режим подключения: **"Передача файлов"** (MTP) или **"Передача фото"** (PTP)

## Шаг 3: Запустите приложение из Android Studio

1. Откройте проект в **Android Studio**
2. В верхней панели (рядом с кнопкой Run ▶️) выберите ваше устройство из списка
   - Если устройства нет в списке, нажмите на выпадающий список и выберите **"USB Device"**
3. Нажмите кнопку **Run** (▶️) или нажмите `Shift + F10`
4. Приложение автоматически:
   - Соберется
   - Установится на телефон
   - Запустится

## ✅ Готово!

Приложение должно запуститься на вашем телефоне.

## 🔧 Если телефон не виден в Android Studio:

### Вариант 1: Установите USB драйверы
- Для **Samsung**: установите Samsung USB Driver
- Для **Xiaomi**: включите "Отладка по USB" и "Разрешить отладку через USB (безопасность)"
- Для **Huawei**: включите "Отладка по USB" и "Разрешить установку через USB"
- Для других брендов: установите драйверы с сайта производителя

### Вариант 2: Проверьте подключение
```bash
# Откройте терминал/командную строку и выполните:
adb devices
```

Если видите ваше устройство в списке - всё хорошо!
Если нет - проверьте:
- USB кабель (должен поддерживать передачу данных, не только зарядку)
- Режим подключения (должен быть "Передача файлов")
- Включена ли "Отладка по USB"

### Вариант 3: Альтернативный способ (через APK)

Если USB не работает:

1. **Соберите APK** в Android Studio:
   - **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
   - Дождитесь завершения
   - Нажмите **"locate"** в уведомлении

2. **Скопируйте APK на телефон**:
   - Через USB (в режиме "Передача файлов")
   - Или через облако (Google Drive, Dropbox)
   - Или через email

3. **Установите APK на телефоне**:
   - Откройте **Файлы** → найдите `app-debug.apk`
   - Нажмите на него
   - Если появится предупреждение, разрешите установку из неизвестных источников
   - Нажмите **"Установить"**

---

## 📱 Где найти APK файл:

После сборки APK будет в:
```
app\build\outputs\apk\debug\app-debug.apk
```

---

## ⚠️ Важно:

- **Минимальная версия Android**: 7.0 (API 24)
- При первой установке может потребоваться разрешение на установку из неизвестных источников
- Для разработки используйте **Debug APK** (не требует подписи)
